### 异常的分类

> [!abstract] 异常分为两大类：受检异常和非受检异常
> - 受检异常：在编译时期就必须处理
> 	`Class forName = Class. forName ("java. lang String");`
> 
> 
> - 非受检异常：在编译时期不必须处理（发生在程序运行时）

### 异常的体系

最顶级的父类Throwable

`Throwable throwable = new Throwable();`

> [!abstract] Throwable之下两个子类
> 
> - Error：严重错误，一般不会发生，如：内存溢出（一旦发生，不可拯救）
> - Exception：一般性错误，可以处理，（例如小感国）


1. **ArrayIndexOutofBoundsException**
	```java
	int[] arr = {1, 2, 3, 4, 5};
	System.out.println(arr[10]);
	```
2. **`NullPointerException`（空指针异常）**
	```java
	String a = null;
	int length = a.length(); // 💥 灾难发生！
	```


```java
public class ExceptionDemo {
    public static void main(String[] args) throws Exception {
        // 方式一：捕获异常
        try {
            // 可能出现异常的代码
            int i = 0;
            int j = 10 / i;
            System.out.println(j);
        } catch (ArithmeticException e) {
            // 处理异常的代码
            System.out.println(e);
        } catch (Exception e) {
            // 处理异常的代码
            System.out.println(e);
            throw e;
        } finally {
            // 最终这里的代码都会执行
            System.out.println("最终这里的代码都会执行");
        }
    }
}
```
